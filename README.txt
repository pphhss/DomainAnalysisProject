
## install

-------This system is based on NodeJS,MySQL-------

1. install NodeJS

2. install MySQL

3. modify "project root directory"/config.js

4. execute MySQL cmd. And copy text of "project root directory"/install/db_schema.sql. And paste copied text into MySQL cmd.

5. execute "npm install".

6. execute "npm start" in project root directory.

7. move http://localhost:"YOUR PORT"
